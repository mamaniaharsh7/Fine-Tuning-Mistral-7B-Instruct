# Fine-Tuning-Mistral-7B-Instruct
Fine Tuning LLM for Short, Context-based Story Generation

- Fine-tuned Mistral-7B-Instruct using QLoRA and 4-bit quantization on 5k+ stories, reducing memory by 75% and achieving BERTScore F1 of 0.845, perplexity of 8.22.
- Created evaluation framework combining automated metrics (BLEU, ROUGE, BERTScore) with manual assessments, scoring 8.96/10 for coherence and 9.05/10 for creativity.
